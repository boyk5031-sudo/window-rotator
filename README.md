# 🖥️ Window Rotator (180°)

A simple Python tool that allows you to:
- Select any running window
- Capture its screen
- Rotate it by **180°**
- Display the rotated output in a new window

---

## 🚀 Features

- ✅ Easy window selection  
- ✅ One-click 180° rotation  
- ✅ Simple GUI using Tkinter  
- ✅ Screenshot-based processing  

---

## 🛠️ Tech Used

- Python  
- Tkinter (GUI)  
- pygetwindow  
- pyautogui  
- Pillow (PIL)  

---
